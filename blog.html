Understanding HTML Semantics

Written by Haytham Naguib
May 5, 2024

HTML semantics help developers write more meaningful and accessible websites. Tags like <article> and <section> add structure to your code.

Here are the main semantic elements you should know:

  • <header>
  • <main>
  • <section>
  • <article>
  • <footer>

For more information, check out this MDN Web Docs article.
